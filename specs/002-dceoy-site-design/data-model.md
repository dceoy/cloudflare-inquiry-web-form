# Data Model: UI Theme Tokens

## ThemeTokens

- **fontFamily**: Ubuntu, fallback to system sans-serif.
- **fontSizes**: base, small, heading sizes for hierarchy.
- **colors**: foreground, muted text, surface, border, accent.
- **spacing**: base spacing units for layout and field gaps.
- **radius**: subtle rounding for inputs/buttons.

## Notes

No persistent data changes are required; tokens map to CSS variables only.

# Quickstart: Dark mode UI

## Verify locally

1. `pnpm dev`
2. Open the app and locate the theme toggle in the header.
3. Toggle light/dark and confirm:
   - Page background, panel, inputs, and buttons update.
   - Text remains readable.
4. Reload the page and confirm the last selected theme persists.
5. Clear localStorage and set your OS to dark, reload, and confirm initial theme is dark.
